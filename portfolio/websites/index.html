State

A social network for sharing opinions.

Work on various back-end, front-end and architecture design aspects of the system, as part of a small but expanding team in a startup. My work included:

Front end

  • HTML 5
  • Responsive pure CSS layout accommodating small screens, built with Sass
  • JavaScript for a richly interactive user experience, using an in-house dynamic templating system (similar in concept to React, Angular, etc)

Back end

  • Development of back-end services using JavaScript (NodeJS) and Ruby (Rails)
  • Design and documentation of a REST API connecting servers, web and mobile applications
  • Design and implementation of a moderation system, connecting to a third party moderator interface (Crisp NetModerator)
